# gld – Great Lorem Distributor

Generate placeholder text in style and volume, effortlessly.

## ✨ Features

- Generate sentences, paragraphs, or entire sections of placeholder text
- Customizable length and word count
- CLI or programmatic usage

## 📦 Installation

```bash
npm install gld
```

## 📖 Usage

```js
import { generate } from 'gld'

generate('Lorem ipsum dolor sit amet')
```
